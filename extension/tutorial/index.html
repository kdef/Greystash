Greystash

Secure Passwords Made Easy

Created by Team Failing Street of the University of Portland

Here's the Deal...

We like to make our passwords short and easy to remember. And we like to use the same one for everything.

Here's the problem...

Short, memorable passwords are easy for the bad guys to figure out. And if you use the same one for everything they can get a hold of everything you do online.

But how else will I be able to remember my password?

I have one word for you...

Greystash

Transforming your weak, human passwords into computer generated ones.

Here's How It Works

Greystash takes a security phrase you set up, the address of the website you are visiting, and your normal easy to remember password and...

  • smushes them all together
  • performs some sophisticated math
  • spits out a unique, random looking password

All You Have to Do Is:

  • Type your easy to remember password
  • That's it! We'll do the rest!

Example

Easy password: cats4ever

Transformed into: 3@JmeK-%`sjRtNl[w

Greystash will put a checkmark in each password field on supported websites. When the check is present, Greystash will work its magic.

If you click on the checkmark, it will change into an 'X', meaning Greystash will be turned off and won't change the password.

Want to create a new online account? Easy!

Simply make sure there is a check in the password fields and your new account will be set up with a Greystash secure password!
